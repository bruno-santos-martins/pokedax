Sobre o Projeto

Uma Pokédex moderna com busca, lista paginada e detalhes completos, construída para aprender e demonstrar boas práticas com Angular.

Visão Geral

A Pokedax integra com a PokéAPI para listar Pokémons, permitir busca por nome e exibir detalhes em um modal com abas: About (altura, peso e habilidades) e Evolution (cadeia evolutiva completa).

Arquitetura

  • Angular 13 + TypeScript como base do app
  • RxJS para chamadas assíncronas e composição de fluxos
  • Serviço único (pokemon.service.ts) centraliza chamadas (lista, detalhes, evoluções)
  • ShareModule com componentes reutilizáveis (navbar, tabela, card, modal, autocomplete)
  • Estilos SASS e utilitários de cor por tipo
  • Roteamento: / (Home) e /sobre (esta página)

Fluxo de Dados

  1. Home carrega a página de Pokémons via serviço (paginação)
  2. Autocomplete consulta nomes para busca rápida
  3. Ao clicar num card, o modal é aberto com About + Evolution
  4. Cores dinâmicas são aplicadas conforme os types do Pokémon

Stack Técnica

Angular 13TypeScriptRxJSSASSPokéAPI
