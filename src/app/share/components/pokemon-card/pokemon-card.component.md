# PokemonCardComponent

Exibe um card visual de um Pokémon, com nome, número, tipo, imagem e stats.

## Selector
```html
<app-pokemon-card [data]="pokemon"></app-pokemon-card>
```

## Inputs
| Nome   | Tipo    | Descrição                |
|--------|---------|--------------------------|
| data   | object  | Dados do Pokémon         |

## Outputs
Nenhum.

## Uso
Ideal para exibir cards em grids ou listas.
